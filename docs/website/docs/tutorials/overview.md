# Tutorials

Learn RaisinDB through hands-on, practical tutorials. Each tutorial is designed to teach you a specific skill while building something real.

## How These Tutorials Work

Each tutorial follows a **learn-by-doing** approach:
- **Copy-paste ready code** - Every code snippet is tested and works
- **Progressive complexity** - Start simple, add features step by step
- **Real-world scenarios** - Based on actual use cases from the community
- **Branches for experimentation** - Use RaisinDB's branching to safely try variations

## Available Tutorials

| Tutorial | What You'll Build | Time | Key Concepts |
|----------|-------------------|------|--------------|
| [Quickstart](/docs/tutorials/quickstart) | A task management app | 15 min | NodeTypes, CRUD, basic queries |
| [News Feed](/docs/tutorials/news-feed) | Content platform (JS/Java/PHP) | 45 min | Hierarchy, GRAPH_TABLE, REFERENCES, RLS |
| [IoT Dashboard](/docs/tutorials/iot-dashboard) | Real-time sensor monitoring | 30 min | Time-series data, branches, WebSocket |
| [Shift Planner](/docs/tutorials/shift-planner) | Employee scheduling system | 45 min | Relationships, constraints, branches for planning |

## Prerequisites

Before starting any tutorial, ensure you have:

- [ ] RaisinDB running locally ([Installation Guide](/docs/getting-started/installation))
- [ ] A REST client (curl, Postman, or similar)
- [ ] Basic understanding of JSON and REST APIs

## Choosing a Tutorial

**New to RaisinDB?** Start with the [Quickstart](/docs/tutorials/quickstart) - it covers the fundamentals in 15 minutes.

**Want to explore advanced features?** The [IoT Dashboard](/docs/tutorials/iot-dashboard) tutorial demonstrates real-time data handling and branch-based analysis.

**Building collaborative workflows?** The [Shift Planner](/docs/tutorials/shift-planner) shows how to use branches for planning, approval workflows, and conflict resolution.
